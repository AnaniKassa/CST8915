## 2. Order-service description
This service is the backbone of the pet store where orders are received from the store-front and trigger at the same time the RabbitMQ message queue.

## 3. product-service
This service is responsible for serving the product catalog, which includes a list of products that can be fetched via a RESTful API.

## 4. store-front
The store-front is the frontend of the pet app that allows users to interact with the app by selecting products and place orders.
