2-Order-service description
This service is a backend service that receives orders from the store-front and sends these orders to a RabbitMQ message queue.

3-product-service
It is responsible for serving the product catalog, which includes a list of products that can be fetched via a RESTful API.

4-store-front
The store-front is the Vue.js frontend that allows users to select products and place orders.
